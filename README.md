# Altersverifikation — Age gate for WordPress & WooCommerce

A configurable **age gate** for WordPress: a full-screen confirmation overlay (simple
confirm or date-of-birth) shown before visitors can access your site or specific content.
Built for age-restricted shops (alcohol, vape, cannabis, adult). **GDPR-clean** — the date
of birth is evaluated entirely in the browser; nothing personal is sent to the server.

## ▶ Live demo (no install)

**[Open the demo in WordPress Playground](https://playground.wordpress.net/?blueprint-url=https://raw.githubusercontent.com/kipphard/altersverifikation/main/demo/blueprint.json)**
— a full WordPress with the age gate active, running entirely in your browser.

## Install

Download the latest [release](https://github.com/kipphard/altersverifikation/releases) zip
→ WordPress admin → *Plugins → Add New → Upload Plugin → Activate*. Configure under the
**Altersverifikation** menu.

## Features (free)

- Full-screen age gate — simple confirmation or date-of-birth
- Site-wide or per-page scope; remember-confirmation cookie
- Configurable text, colours, logo; decline → message or redirect
- GDPR-clean: no personal data leaves the browser (fail-safe — content stays hidden if JS fails)
